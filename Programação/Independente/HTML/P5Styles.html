HTML Styles - CSS

CSS stards for Cascading Style Sheets. It can control the layout of multiple web pages all at once.

Whats is CSS?

CSS is used to format yhe layout of a webpages.

With CSS you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background imagens or background color are to be used, different displays for different devices and screen sizes, and much more!

TIP: the word Cascading means that a style applied to a parents elemts will also apllied to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and other text elements, within the body will also get the same color(unless you specify somthing else)!

Using CSS

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using a <style> element in the <head> section
  • External - by using a <link> element to link to an external CSS

The most common way to add CSS, is to keep the style in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for to try it yourself.

Inline CSS

An Inline CSS, is used to apply a unique style to a single HTML element.

An Inline CSS uses the style attribute of a HTML element.
