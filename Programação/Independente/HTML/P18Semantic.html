HTML Semantic Elements

Semantic Elements = Elements with a meaning


HTML <section> Element

According W3C's HTML documentations: "section is a thematic grouping of content, typically with a heading."

Examples of where a <section> elemenet can be used:

  • Chapter
  • Introduction
  • News items
  • Contact information

A web page could normally be split into sections for introdution, content, and contact information

HTML <article> Element

The <article> element specifies independent, self-contained content.

An article should make sense on its own, and it should be possible to distribute it independently from the rest of the webpage.

Examples of where the <article> element can be used:

NOTE: Use CSS to style elements
  • Forum posts
  • Blog posts
  • User comments
  • Product cards
  • Newspaper articles
