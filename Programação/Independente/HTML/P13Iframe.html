HTML Iframes

AN HTML Iframe is used to display a web page winthin a web page.

HTML Inframe Sytax

The HTML <inframe> tag specifies an inline frame. An inline frame is used to embed another document within thw current HTML document.

Syntax

<iframe src="url" title="description>"</inframe>

TIP: It is a good practice to always include a title attribute for the <iframe>. This is used by screen readers to read out what the content of the iframe is.
