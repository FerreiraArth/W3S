HTML JavaScript

JavaScript makes HTML pages more dynamic and interactive.

The HTML <script> tag is used to define a client-side script(JavaScript).
The <script> element either contaisn script statements, or it points to an external script file throungh the src attribute.
Commona uses for JavaScript are image manipulation, form validation, and dynamic changes of content.
To select an HTML element, JavaScript most often uses the document.getElementById() method.

The HTML <noscript> Tag

The HTML <noscript> tag define an alternate content to be displayed to users that have disabled scripts in their browser or have a browser that doesn't support script.
