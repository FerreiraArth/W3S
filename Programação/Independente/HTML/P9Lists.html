HTML Lists

HTML lists allow web developers to group a set of related items in lists.

Description of List Tags
TAG Description
<ul> Defines an unordered list
<ol> Defines an ordered list
<li> Defines a list item
<dl> Defines a description list
<dt> Defines a term in a description list
<dd> Describes the term in a description list

List Style Type

Unordered List

By default, the unordered list will be marked with bullets (small black cicle).

The CSS list-style-type property id used to define the style of the list item marker. It can have on of the following values for unordered list:

Unordered List
Value Description
disc Sets the list item marker to a bullet(default)
circle Sets the list item marker to a cicle
square Sets the list item marker to a square
none The list items will not be marked

NOTE: A list item (<li>) can contain a new list, and other HTML element, like images and links, etc.
