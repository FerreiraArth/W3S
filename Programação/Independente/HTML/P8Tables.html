HTML Tables

The HTML tables allow Web developers to arrange data into rows and colomns. Being composed of, <th>(table head), <tr>(table rows), <td>(table datas).

By default, the text in <th> elements are bold and centered, but you can change that with CSS.

Zebra Stripes - horizontal

If you add a background color on every other table row, you will get a nice zebra stripes effect.

To style every other table row element, use the :nth-child(even) selector like this:

Note: If you use (odd) instead of (even), the styling will occur on row 1, 3, 5 etc. instead of 2, 4, 6 etc.

Zebra Stripes - Vertical

To make vertical zera stripes, style every other column, instead of every other row.

Set the :ntd-child(even) for table data elements.

Note: Put the :ntd-child() selector on both th and td elements if you want ti have the styling on both headers and regular table cells.

Combine Vertical and Horizontal Zebra Stripes

You can combine the styling from the two examples above and you will have stripes on every other row and every other column.

TIP: If you use a tranparent color you will get an overlapping effect. Use the rgba() color to specify the tranparency of the color.

Horizontal Dividers

Just add the border-bottom property to all tr elements to get horizontal dividers.

Hoverable Table

Use the :hover selector on tr to highlight table rows on mouse over.
