Color Values

In HTML colors can also be specified using RGB value, HEX values, HSL values, RGBA values, and HSLA values.

The following three <div> elements have their background color set with RGB, HEX, and HSL values:

#8a2be2"

The following two <div> elements have their backfround color set with RGBA and HSLA values, which add an Alpha channel to the color (here we have 50% transparency)

rgba(138, 43, 226)

hsl(271, 76%, 53%, 0.5)
