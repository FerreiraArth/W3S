CSS Borders

Borders - Individual Sides

In CSS, there are also properties for specifying each of the borders (top, rigth, bottom, and left):

Example:

            p {
                border-top-style: dotted;
                border-right-style: solid;
                border-bottom-style: dotted;
                border-left-style: solid;"
            }

Result:

Different borders styles

So, here is how it works:

  • If it has four values: border-style: dotted solid double dashed;
  • If it has three values: border-style: dotted solid double;
  • If it has two values: border-style: dotted solid;
  • If it has one values: border-style: dotted;
