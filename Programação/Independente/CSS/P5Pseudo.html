CSS Pseudo-Classes

A Pseudo-Classes is used to define a special state of an element. It can be used to:

  • Style an element when a user mouser over it
  • Style visited and unvisited link differently
  • Style an element when it gets focus

Syntax

            selector:pseudo-class {
                property: value;
            }
