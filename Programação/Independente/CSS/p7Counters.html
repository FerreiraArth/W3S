CSS Counters

Automatic Numbering With Counters

CSS counters are like "variables". The variable values can be incremented by CSS rules (Which will track how many times they are used).

To work with CSS counters we will use the following properties:

  • counter-reset - Creates or resets a counter
  • counter-increment - Increments a counter value
  • counter - Inserts generated content
  • counter() or counters() fuction - Adds the value of a counter to an element

To use the CSS counter it must first be created with counter-reset.

Counter in outlined list

A counter can also be useful to make outlined lists because a new instance of a counter is automatically created in child elements. Here use the counters() function to insert a string between different levels of nested counters:

Example

                ol {
                    counter-reset: section;
                    list-style-type: none;
                }

                li::before {
                    counter-increment :section;
                    content: counters(section, ".") " ";
                }
